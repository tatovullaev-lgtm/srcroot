bmnroot/base/

The collection of base classes for the BmnRoot framework.

* *cbm* - base classes from the CBM project at FAIR used in BmnRoot
* *data* - data storage classes
* *header*- header classes for output files at the BM@N processing steps
* *math* - base math tools to be used while processing
* *raw* - raw data storage classes
* *source* - classes for handling input files
