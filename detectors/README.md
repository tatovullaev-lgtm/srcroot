\defgroup detectors Detector Simulation

bmnroot/detectors/
==================

A set of classes to prepare Monte Carlo simulation of BM@N detectors (including classes for detector's points, hits and hit producer)

Classes for MC simulation of:
* *armTriggers* - Arm Trigger Detectors
* *bc* - Beam Counter
* *bd* - Barrel Detector
* *csc* - Cathode Strip Chambers
* *dch* - Drift Chambers
* *ecal* - Electromagnetic Calorimeter
* *FD* - Forward Detector
* *fhcal* - Forawrd Hadron Calorimeter
* *gem* - Gas Electron Multipliers
* *hodo* - Hodoscope
* *mwpc* - Multi Wire Proportional Chambers
* *ndet* - Neutron Detector
* *scwall* - Scintillation Wall
* *SiBT* - Silicon Beam Tracker
* *silicon* - Silicon Detector
* *SiMD* - Silicon Multiplier Detector
* *SiProbe* - Silicon Probe Detector
* *SiProf* - Silicon Profilometer
* *ssd* - Silicon Strip Detector 
* *sts* - Silicon Tracking System
* *tof* - TOF-700: the farthest of two Time-of-Flight Detectors
* *tof1* - TOF-400: the nearest of two Time-of-Flight Detectors
* *tof701* - new implementation of the TOF-700 detector
* *vacwall*
* *vsp* - Vertex Silicon Plane (high-precision microstrip coordinate detector of the inner tracking system)
* *zdc* - Zero-Degree Calorimeter
