/**
\defgroup database Database Interfaces

bmnroot/database/
=================

C++ API classes to work with the Information Systems of the BM@N experiment

* *elog_db* - API for the Electronig Logbook Platform (e-Log)
* *event_db* - API for the Event Metadata System (EMS)
* *geo_db* - API for the Geometry Information System (Geometry Database)
* *uni_db* - API for the Unified Condition Database (UniConDa)
* *examples* - example for working with the database interfaces, which are specific to the BM@N
* *tango* - C++ interface and macros to work with the Tango database used by the Slow Control System
* *tools* - inner auxillary tools for developers of the interfaces

 */
