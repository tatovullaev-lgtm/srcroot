/**
\defgroup macro Macros

bmnroot/macro/
==============

Collection of implemented macros in the BmnRoot framework.

They are groupped in the following directories:
* *alignment* - macros to perform alignment
* *cbm* - macro files with CBM classes
* *detectors*- set of macros divided by detector subdirectories to check processing in the corresponding detector
* *embedding* - macro files for embedding
* *eventdisplay* - event display macro file and additional necessary files to run Event Display
* *geometry* - set of macros to produce detector geometry files, mainly in CERN ROOT format
* *globaltracking* - macro files for testing global tracking
* *howTo* - HowTo macros to provide users with a list of examples
* *magfield* - set of macros to create and work with magnetic field of the experiment
* *miscellaneous* - directory with macros to perform some miscellaneous actions
* *monitor* - macros and auxillary files to run Online Histogramming System for monitoring data quality during BM@N Runs
* *physics* - set of macros to perform physics analysis of the BM@N data
* *profiling* - files for profiling main BmnRoot macros and checking them for memory leaks and errors
* *raw* - some macro files to work with Raw Data Decoder of the BM@N experiment
* *recotools* - tools concerning event reconstruction
* *run* - main macro files of the BmnRoot package to run simulation, reconstruction and others
* *steering* - set of files for steering

 */
